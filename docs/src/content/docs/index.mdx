## Features

<CardGrid stagger>
	<Card title="10+ Sources" icon="setting">
		Play music from YouTube, SoundCloud, Twitch, Spotify, Apple Music, direct links, and local files.
	</Card>
	<Card title="Interactive TUI Dashboard" icon="document">
		Monitor resources, queue length, active players, and logs in real-time through a beautiful terminal interface.
	</Card>
	<Card title="Spotify Integration" icon="open-book">
		Full support for Spotify tracks, playlists, and curated mixes using SpotifyScraper to bypass API limits.
	</Card>
	<Card title="Self-Hosted & Private" icon="pencil">
		All logs stay local to your machine. No telemetry is sent anywhere. No web UI, no paywalls.
	</Card>
</CardGrid>
