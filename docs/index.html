Important Update

The Public Bot Has Been Discontinued

Due to increasingly aggressive restrictions from Google and YouTube, the public instance of Playify was constantly being detected and blocked. Maintaining a bot serving thousands of servers became impossible under these conditions — every fix was met with a new block within days.

After months of fighting these bans, I made the difficult decision to discontinue the public bot. But Playify itself is alive and thriving — it has simply evolved into a fully self-hosted project.

The Solution: Self-Hosting

By running your own private instance, your bot flies under the radar. No more bans, no more interruptions. You get full YouTube support, complete privacy, and total control over your music experience. It takes less than 5 minutes to set up.

KEY FEATURES

Full Spotify Integration

Play any Spotify playlist, including official curated ones like "This Is" and "Radar" that most bots can't handle.

Transform your music with the /filter command to apply effects like slowed, reverb, nightcore, and more.

Windows Easy Setup

Easiest

A simple startup script (start.bat) that handles everything — Python, FFmpeg, dependencies. Just enter your API keys and you're done.

Docker

Recommended

Clone, configure your .env file, and run docker compose up -d. Works on any OS with Docker installed.

Manual Setup

For developers who want full control. Install Python 3.9+, FFmpeg, clone the repo, and run python playify.py.

GET STARTED IN MINUTES

1

Create a Bot

Head to the Discord Developer Portal and create a new application with a bot token.

2

Get API Keys

Grab your Spotify Client ID/Secret and optionally a Genius API token for lyrics.

3

Choose Your Method

Use the Windows Easy Setup (start.bat), Docker, or manual Python setup — whatever suits you best.

4

Enjoy the Music

Invite your bot to your server and type /play. That's it — you're done!
